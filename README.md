# 🔎 Avis de recherche

Ce dépôt permet de publier et suivre des avis de recherche via les Issues GitHub.

## ➤ Comment créer un avis

1. Clique sur "New Issue"
2. Sélectionne "Avis de recherche"
3. Remplis les informations

## ➤ Statuts

- 🟢 Ouvert : personne recherchée
- 🔴 Fermé : personne retrouvée

## ⚠️ Avertissement

Ce projet est destiné à un usage éducatif ou fictif.
Merci de respecter la vie privée.
